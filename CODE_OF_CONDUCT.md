# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as
its code of conduct. The canonical text lives at:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

We follow it as written. Pinning by URL rather than inlining the
text keeps us aligned with future revisions of the Covenant
without manual edits.

## Reporting

To report a violation:

1. Use the same private channel documented in [`SECURITY.md`][sec]
   (GitHub private vulnerability reporting, or the contact
   email noted there).
2. Include enough detail for us to act on the report.
3. Reports are read by maintainers only, kept confidential, and
   handled per the enforcement steps in the Covenant.

We aim to acknowledge within 3 business days.

[sec]: ./SECURITY.md

## Scope

The Code of Conduct applies in all project spaces — issue
tracker, pull requests, discussions, and any other forum where
someone is participating as a contributor or representative of
the project.

## Enforcement

Maintainers may, at their discretion, warn, edit/remove
inappropriate content, restrict participation temporarily, or
ban a contributor permanently. The criteria and process are
those described in the Covenant.
